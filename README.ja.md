# imgemb

画像ファイルをBase64エンコードされたデータURLに変換し、HTMLの `<img>` タグに埋め込める形式にするシンプルなクライアントサイドツールです。これにより、画像ファイルを別途用意することなく、ソースコードに直接画像を埋め込むことができます。

## デモ

**[ブラウザで試す](https://code4fukui.github.io/imgemb/)**

画面は、画像ファイルをドロップする点線のエリアと、生成されたHTMLタグを表示するテキストエリアで構成されています。

## 機能

- **ドラッグ＆ドロップ操作:** 画像ファイルをブラウザにドラッグ＆ドロップするだけです。
- **即時変換:** Base64エンコードされた `data:` URL をその場で生成します。
- **HTMLタグ出力:** 変換結果を完全な `<img src="...">` タグで自動的に囲みます。
- **完全なクライアントサイド処理:** すべての処理はブラウザ内で完結します。画像がサーバーにアップロードされることはありません。
- **自己完結型:** ツール全体が単一のポータブルなHTMLファイルで構成されています。

## 使い方

1. [デモページ](https://code4fukui.github.io/imgemb/)を開きます。
2. パソコンから画像ファイルをドラッグし、点線のエリアにドロップします。
3. 完全な `<img src="...">` タグが下のテキストエリアに表示されます。
4. 生成されたHTMLをコピーして、プロジェクトに貼り付けます。

## 必要条件

`FileReader` APIをサポートするモダンブラウザ（例: Chrome、Firefox、Safari、Edge）。
